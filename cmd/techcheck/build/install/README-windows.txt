First-time launch on Windows

This app isn't signed with a code-signing certificate yet, so when
you double-click it Windows will show "Windows protected your PC"
with "Microsoft Defender SmartScreen prevented an unrecognized app
from starting."

To allow it:

1. Click "More info" on the SmartScreen dialog.
2. Click "Run anyway."

After this first time, Windows will remember and let it open
normally.
