## What's new in 0.1.3

### Cosmetic fixes

- The update banner and release-notes modal now use the Sunrise dark theme rather than a jarring white panel. Long release notes scroll inside the modal instead of pushing the Download button off the screen.
- The Windows `.exe` and macOS `.app` carry the Sunrise app icon again (regressed accidentally in 0.1.2 during a build-config reorg).

No functional changes. Reports from 0.1.3 are wire-compatible with 0.1.2 and 0.1.1.
