# Sunrise Tech Check 0.1.7

## What's new

- **Windows now reports displays, GPUs, and connected USB devices.** Previous Windows builds returned empty lists for these — they were only populated on macOS. Tech Check on Windows now enumerates monitors (width, height, refresh rate), the graphics adapter (model + VRAM), and USB peripherals (name, manufacturer, vendor/product ID), so reports give Sunrise the same hardware picture as macOS does.

## Internal

- New `peripherals_windows.go` (`//go:build windows`) replacing the non-darwin stub for displays/GPUs/USB. Linux remains stubbed.
- Mirrors the macOS shell-out pattern: PowerShell calls to `Win32_VideoController` + `System.Windows.Forms.Screen.AllScreens` for displays and GPUs, and `Win32_PnPEntity` filtered to USB devices.
- Sets `[Console]::OutputEncoding = UTF8` on each PowerShell call so non-ASCII device/manufacturer names survive intact.
- Hides the cmd window on the new shell-outs via the existing `hideWindow` helper, consistent with the security/firewall PowerShell calls added in 0.1.4.
- 9 new unit tests cover the JSON parsers, including multi-monitor and duplicate-resolution edge cases.
