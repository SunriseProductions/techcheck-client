# Sunrise Tech Check 0.1.5

## What's new

- **First-launch instructions ship with the download.** macOS DMGs now include a `Read Me First.txt` next to the app, plus an Applications-folder symlink, against a Sunrise-branded background showing the drag-to-install layout. Windows downloads are now a ZIP containing the EXE and a matching `Read Me First.txt`. Both READMEs explain what the system will say about the unsigned binary (macOS Privacy & Security, Windows Defender SmartScreen) and how to allow it — until we ship code-signed builds.

## Internal

- DMG packaging switched from `hdiutil create` to `create-dmg` for layout / background / drop-link affordances. Windows packaging adds a `zip` step. Install assets (READMEs + DMG background) live at `cmd/techcheck/build/install/`.
- The Windows artifact filename in the app-registry changes from `*-x64.exe` to `*-x64.zip`. Existing 0.1.4 download links are unaffected.
